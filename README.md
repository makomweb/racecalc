# Race Calc - Pace Time Distance

A fast and intuitive calculator for converting between pace, time, and distance.

Perfect for runners, cyclists, and anyone tracking performance metrics.

**Live Site:** https://makomweb.github.io/racecalc

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (comes with Node.js)

## Getting Started

```bash
# Install Dependencies
npm install

# Run Development Server
npm run dev

# Run Tests
npm run test

# Build for Production
npm run build

# Lint Code
npm run lint
```

Open browser at: http://localhost:5173
